# Firebase Authentication Login Setup (React + Vite + Tailwind + Shadcn UI)

This project is a simple and modern authentication system built with **Vite + React**, styled using **Tailwind CSS and Shadcn UI**, and powered by **Firebase Authentication**.  
Users can **sign up**, **log in with Google**, **reset passwords via spam email**, and the project is **hosted on Netlify**.

---

##  Features
-  Create Account (Email & Password)
-  Login with Google
-  Password Reset via Email
-  Beautiful UI using Shadcn + Tailwind
-  Deployed on Netlify
-  My github profile
